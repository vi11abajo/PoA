# Постоянные указания для Claude Code

## Основные правила
- Используйте Python 3.8+
- Следуйте PEP 8 для стиля кода
- Добавляйте docstrings к функциям и классам
- Используйте type hints

## Структура проекта
- `src/` - основной код
- `tests/` - тесты
- `docs/` - документация

## Команды для запуска
- Тесты: `python -m pytest`
- Линтер: `flake8`
- Форматирование: `black .`
